# Retail Sales Analytics

## Overview

This project analyzes a retail sales dataset using SQL (SQLite) to answer common business questions related to sales performance, customers, products, and regional trends.

The analysis includes data exploration, sales analysis, customer insights, product performance, regional analysis, and basic SQL window functions.

## Dataset

- Sample Superstore Dataset
- Records include orders, customers, products, sales, profit, discounts, and shipping information.

## Objectives

- Explore the retail dataset
- Analyze sales and profit trends
- Identify top-performing customers and products
- Compare regional performance
- Practice SQL for business data analysis

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, AVG, COUNT)
- HAVING
- LIMIT
- Subqueries
- Window Functions (RANK, Running Total)
- Date Functions (strftime)
